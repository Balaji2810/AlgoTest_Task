AlgoTest Task API Endpoints

 

Login API

This API is used to login to the service by using phone number or email along with the password

POST https://algotesttask.onrender.com/api/v1/token/refresh

form-data

phone_or_email  :  PhoneOrEmailRegex

password  :  PasswordRegex

 

Access Request API

This API is used to request for access token

POST          https://algotesttask.onrender.com/api/v1/token/access

header

Authorization  :  Bearer <Refresh Token>

 

SignUp API

This API is used to create a new user

POST          https://algotesttask.onrender.com/api/v1/user/signup

form-data

phone  :  PhoneRegex

email  :  EmailRegex

email_otp  :  str

phone_otp  :  str

name  :  NameRegex

password  :  PasswordRegex


Forgot Password API

This API is use to reset the password

PUT https://algotesttask.onrender.com/api/v1/user/password

form-data

phone_or_email  :  PhoneOrEmailRegex

new_password  :  PasswordRegex

otp  :  str

 

Check User API

This API is used to check whether the User is registered in the database or not

POST          https://algotesttask.onrender.com/api/v1/user/check

form-data

phone_or_email  :  PhoneOrEmailRegex

 

OTP Email API

The APi will send a OTP to the mentioned email

POST          https://algotesttask.onrender.com/api/v1/otp/email

form-data

          email  :  EmailRegex

 

OTP Email Verify API

This API will verify the OTP from email

POST          https://algotesttask.onrender.com/api/v1/otp/email/verify

form-data

          email  :  EmailRegex

          otp  :  str

 

OTP Phone API

The API will send a OTP to the mentioned phone

POST          https://algotesttask.onrender.com/api/v1/otp/phone

form-data

          phone  :  PhoneRegex

 

OTP Phone Verify API

This API will verify the OTP from phone

POST          https://algotesttask.onrender.com/api/v1/otp/phone/verify

form-data

          phone  :  PhoneRegex

          otp  :  str

 

Save Records API

This API will save all the records under the user

POST          https://algotesttask.onrender.com/api/v1/data/

header

          Authorization  :  Bearer <Access Token>

form-data

          data  :  JSONstring(list)

Get All Records API

This API will fetch all the records of the user

GET   https://algotesttask.onrender.com/api/v1/data/

header

          Authorization  :  Bearer <Access Token>

 

Get Single Record

This API will fetch a single record of the user

GET   https://algotesttask.onrender.com/api/v1/data/<id>

header

          Authorization  :  Bearer <Access Token>

 

Regex

PhoneRegex  : "^[6-9]\d{9}$"

EmailRegex  : "^([\w\.\-]+)@([\w\-]+)((\.(\w){2,63}){1,3})$"

PhoneOrEmailRegex  :  "^([6-9]\d{9}|([\w\.\-]+)@([\w\-]+)((\.(\w){2,63}){1,3}))$"

NameRegex  :  "^[\w\s]{3,}$"

PasswordRegex  :  "^[\w\s]{6,16}$"
